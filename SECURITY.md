# Security Policy

## Supported Versions

The following versions of **Worpen** are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.4.x   | :white_check_mark: |
| 2.3.x   | :white_check_mark: |
| < 2.3   | :x:                |

## Reporting a Vulnerability

**Worpen** takes security seriously. If you discover a vulnerability, please follow these steps:

1.  **DO NOT** open a public issue on GitHub.
2.  Encrypt your findings using our PGP Key (ID: `0xDEADBEEF`).
3.  Email the encrypted report to `security@worpen.io`.

### Response Timeline

*   **Acknowledgement:** 24 hours.
*   **Triage:** 48 hours.
*   **Patch:** 7 days (critical) / 14 days (non-critical).

## Bounty Program

We offer bounties for critical vulnerabilities that affect the integrity of the **Hive Mind** topology.
*   **RCE:** $5,000+
*   **Privilege Escalation:** $2,500
*   **Information Leak:** $500

> *"Trust nothing. Verify everything."*
